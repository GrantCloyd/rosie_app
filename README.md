# README

* Ruby/Rails versions
  - Ruby - 3.2.2
  - Rails - 7.1.2

* System dependencies
  - Postgresql@16
  - tailwind-css 

* Database creation
  - Install and start postgresql service
  - `rails db:create && rails db:migrate && rails db:seed`

* Start App
  - install Ruby 3.2.2
  - run bundle
  - build css - `rails tailwindcss:build`
  - create db/seed if you haven't already
  - `rails server`
  - open app on localhost:3000
